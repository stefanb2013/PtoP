# PtoP
***
The software "Peer-to-Peer NG Runtime" as well as all previous versions was discontinued by B&R Industrie-Elektronik GmbH as of June 30, 2026.
This software will no longer be updated or further developed, maintained, serviced, and/or patched in any way. No bug fixes, security updates, or adaptations to new operating system environments will be provided. Any potential security-related vulnerabilities will likewise no longer be addressed.

In order to give existing users of the software the opportunity to maintain their systems or to continue using the technology in their own projects, the source code of this software has been made available in this repository under the MIT License, with the consent of B&R Industrie-Elektronik GmbH.
